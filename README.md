This project demonstrates a Bank Account Money Transfer System implemented using Oracle SQL. The system manages customer accounts and records money transfer transactions between accounts.
**Project Overview**
The project contains two main database tables:
BANK_ACCOUNT_TBL
Stores customer bank account details.
Includes account ID, customer name, and account balance.
BANK_TRANSFER_TBL
Stores transaction details between accounts.
Includes transaction ID, sender account, receiver account, transaction date, and transfer amount.

**Features**
Create and manage bank account records.
Perform money transfer between accounts.
Automatically update account balances.
Maintain transaction history.
Use SQL JOIN queries to view complete transfer details with customer information.
